AI Face Recognition System

A fully on-premise, real-time face recognition platform for enterprise security and attendance tracking — built end-to-end from model pipeline to backend API to GUI dashboard.

Problem

Enterprises needed secure, real-time identity recognition without cloud API dependency or latency.

Solution

Fully on-premise AI pipeline with scalable backend, multi-camera support, and live alerting.

Technical Architecture

  • Face embedding via InsightFace + ONNX Runtime for fast, accurate identification
  • Cosine similarity matching with configurable confidence thresholds
  • FastAPI backend with PostgreSQL storage and Redis caching for <100ms response times
  • GUI dashboard with real-time feed, attendance logs, and automated alerts
  • Dockerized multi-camera support for scalable enterprise deployment

Stack

Python · FastAPI · PostgreSQL · Redis · OpenCV · InsightFace · ONNX · Docker
